# MovieHub 🎬

MovieHub is a high-end movie discovery and aggregation platform. It allows users to search for movies, watch trailers, and discover the latest releases through a seamless, modern interface.

## 🛠️ Tech Stack
- **Frontend:** Next.js, Tailwind CSS, TypeScript
- **Backend:** FastAPI, Python, Pydantic
- **Database:** PostgreSQL
- **APIs:** TMDB, YouTube Data API

## 📂 Project Structure
- `/moviehub-frontend`: Next.js application.
- `/moviehub-backend`: FastAPI server.
- `/docs`: Documentation and deployment plans.

## 🚀 Getting Started
Check the `/docs` folder for detailed setup and deployment instructions.
